# Excel for Data Analysis

> A personal learning journal documenting what I built and discovered through this course.

---

## About This Course

Coming into this course, I already had a solid foundation in Excel. What this course helped me do was take that familiarity further — learning how to use Excel not just as a spreadsheet tool, but as a proper data analysis environment. Through real-world case studies, I learned how to work with data more intentionally: summarizing it, interrogating it, and presenting it in ways that actually tell a story.

---

## What I Learned

- How to apply **descriptive statistics** to summarize datasets and understand distributions
- How to use **conditional formatting** to visually highlight trends, outliers, and key insights at a glance
- How to write and combine **Excel formulas** to perform meaningful analysis on real data
- How to conduct **customer analysis** and **performance analysis** using structured data
- How to build **research reports** that translate raw numbers into clear, readable findings
- How to present data through **storytelling** — making insights understandable to any audience

## How the Course Was Structured

The course was case-study driven, which I found particularly useful. Rather than learning formulas in isolation, I applied them to realistic business scenarios — analyzing actual datasets, drawing conclusions, and presenting findings in a structured format. It bridged the gap between knowing Excel and using it analytically.

---

## My Takeaway

I came in comfortable with Excel, but this course sharpened how I *think* about data. It's one thing to know your way around a spreadsheet — it's another to know what questions to ask of your data and how to communicate what you find.
