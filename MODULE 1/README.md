# Statistical Functions & Analysis Techniques

This module introduces the foundational statistical functions essential for any data analysis task. Learners will go beyond memorizing syntax to understand when and why to use functions, such as AVERAGE, MEDIAN, and COUNTIF. By drawing on the analysis of 1.1 billion NYC taxi trips and the Bellabeat health-tech case study, learners will practice applying these functions to transform raw data into clear business insights.

### Learning Objectives
- Analyze datasets using core statistical functions (AVERAGE, MEDIAN, STDEV, and COUNTIF) to summarize key characteristics and answer business questions.
- Evaluate a given dataset to select and justify the appropriate statistical functions and formatting techniques needed to extract and present meaningful insights for stakeholder review.
